Secret-Service-Tutorial


Willkommen beim Secret-Service der FHB. Es handelt sich hierbei um einen praktischen Demonstrator des Bell-LaPadula-Sicherheitsmodelles. Das Prinzip des Modells kann man mit zwei Regeln beschreiben:

  1. No-Read-Up: Es darf niedriger eingestuften Personen nicht möglich sein, Informationen von vertrauenswürdigereren Personen zu lesen.
  2. No-Write-Down: Höher eingestufte Personen dürfen nicht in Dateien von weniger vertrauenswürdigen Personen schreiben. Dadurch wird verhindert, dass sie Informationen "nach unten" weitergeben.

Mehr Informationen zum Modell finden Sie unter http://de.wikipedia.org/wiki/Bell-LaPadula
Im unteren Bild sehen Sie die Home- bzw. die Loginseite. Falls Sie schon ein Subjekt erstellt haben, können Sie sich hier einloggen. Andernfalls finden Sie in der oberen linken Ecke den Link zum Dashboard. Das Dashboard dient zum verwalten der Subjekte. Sie können Subjekte erstellen, ändern oder löschen. Zusätzlich können Sie alle Objekte löschen, falls Sie den Demostrator zurücksetzen möchten. Mehr dazu unter Dashboard. In der oberen rechten Ecke finden Sie den Link zu dieser Anleitung :)
